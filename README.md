Запуск ноутбука в Google Colab

1. Загрузить датасет  в корень гугл диска.  
2. Открыть файл `lab1_yolo_colab.ipynb` в colab.  
3. В Colab включить `GPU` : `Runtime` -> `Change runtime type` -> `T4 GPU`.    
4. Запускать ячейки сверху вниз.
